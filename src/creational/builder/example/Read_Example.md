## 단계별 핸드폰 생성
- 이 예시에서의 빌더 패턴은 다양한 핸드폰 모델을 단계별로 생성할 수 있음 
- 이 예시는 빌더가 어떻게 같은 생성 단계들을 사용하여 다양한 종류의 제품들을 생성하는 방법을 보여줌
- 디렉터는 생성 순서를 통제하며 특정 자동차 모델을 생성하기 위해 어떤 특정 생성 단계를 호출해야 하는지를 알고 있음
- 디렉터는 빌더들의 공통 인터페이스를 통해서만 빌더들과 함께 작동하며 이는 다양한 유형의 빌더들을 디렉터에게 전달할 수 있도록 함
- 최종 결과는 객체에서부터 가져옴
  - 디렉터는 결과 제품의 유형을 알 수 없기 때문
  - 빌더 객체만이 자신이 정확히 무엇을 생성하는지 알고 있음
